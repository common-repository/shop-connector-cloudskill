== Description ==

Allow you to connect your Cloudskill account to your Woocommerce Shop and trigger orders directly

== Installation ==

1. Install the Cloudskill Shop Connector through the WordPress.org plugin directory or manual upload.
2. Make sure you got your WooCommerce Shop set up and no password on your site
3. Go to the settings and paste in the Cloudskill key you can receive inside your Cloudskill application to connect to it.
4. Use the Connect API Button in the settings to create an WooCommerce API User and trigger the information to your Cloudskill account
5. Finished! From now on you should receive all orders directly and the article and prices should be syncronized to your shop depending on your settings in your Cloudskill Account
